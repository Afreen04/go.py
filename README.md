# go.py

A text-based go game written in python.

## Usage

    usage: go.py [-h] [-s SIZE]
    
    Starts a game of go in the terminal.
    
    optional arguments:
      -h, --help            show this help message and exit
      -s SIZE, --size SIZE  Size of board.

## Controls
    
    Standard WASD for cursor moving
    SPACE for move
    U for undo
    R for redo
